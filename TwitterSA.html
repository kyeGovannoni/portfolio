Twitter Sentiment Analysis

I wanted to collect tweets to perform sentiment analysis on, to do this I used the Tweepy package with Twitters API and MonkeyLearn API. To take full advantage of Tweepy I created a separate py file in PyCharm, This file contained custom classes so that I could call the classes and return a df of Full_Text tweets.

Below you can find the Tweepy class file and part of the Sentiment analysis, involving prepossessing the tweets and then exporting the sentiment results.
